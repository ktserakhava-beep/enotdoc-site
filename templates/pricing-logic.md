# Pricing Logic

## Waste

### Default Test
- `1 490 BYN setup + 390 BYN/month`

### Backup Test
- `690 BYN paid audit`
- Credit into setup if client proceeds within 7 days

### Use Default When
- 1 legal entity
- 1 site
- typical waste profile
- source docs at least partially available

### Use Backup When
- buyer is cautious
- scope is unclear
- docs are messy
- fit exists but trust is still low

### Price Framing
- First explain scope
- Then explain result
- Then give price
- Never anchor on "instruction alone costs X"

### Discount Rules
- Reduce only for genuinely smaller scope
- Avoid dropping setup below `1 190 BYN`
- Avoid dropping monthly below `290 BYN`

## ROP

### Default Logic
- fixed setup + first quarter package

### Backup
- paid audit / data-map session

### Rule
- no pricing before owner/process/source docs are understood

## Auto

### Default Logic
- pilot setup for 1–2 sites + support

### Rule
- only after waste traction or warm intro
