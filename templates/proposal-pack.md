# Proposal Pack

## Proposal Structure

### 1. Context
- Short summary of client situation
- Why this scope is needed now

### 2. Scope
- 1 legal entity / 1 site by default
- Typical waste compliance contour only

### 3. Deliverables
- Current-state review
- Missing-items list
- Instruction update / preparation
- POD setup logic
- 1-отходы basis
- acts / contracts archive logic
- deadlines calendar
- first monthly support rhythm

### 4. Timeline
- 24h fit-check
- 7–14 days for setup

### 5. What We Need From Client
- current instruction
- latest `1-отходы`
- 2–3 acts / contracts
- one responsible contact

### 6. Price
- setup
- monthly support

### 7. Boundaries
- no emissions/water/ecopassport in default scope
- no complex hazardous edge cases without review
- no autonomous final expert guarantee

### 8. Safety Language
- Based on source docs provided by client
- Drafts and working package go through review
- Non-typical situations may require external expert review

### 9. Next Step
- confirm scope
- send documents
- approve start
