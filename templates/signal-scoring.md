# Signal Scoring

## Signal Levels

### 1–3: Weak
- asks to send information only
- vague interest
- no clear buyer
- no urgency

### 4–6: Medium
- agrees to call
- can describe pain
- buyer partially visible
- no real budget discussion yet

### 7–8: Strong
- asks about scope and timeline
- asks what documents are needed
- asks how much setup costs
- can identify decision-maker

### 9–10: Very Strong
- asks for proposal
- asks to start this month
- provides docs
- discusses price and budget realistically

## Buying Signal Checklist
- clear pain stated
- owner/buyer identified
- urgency visible
- source docs available
- price discussion started
- next step scheduled

## Red Flags
- wants everything at once
- no source docs
- no buyer clarity
- wants "automatic result without review"
- complex scope from day one
